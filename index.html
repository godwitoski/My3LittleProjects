MIS 3 MINI PROYECTOS

Bienvenidos a mis 3 mini proyectos creados en los inicios de mi formación como Programador Web. Son totalmente diferente, 2 de ellos se adaptan a todos los dispositivos, mientras que el juego de Atlantis es para PC para tener la experiencia completa.

  • El buscador de imágenes por color está realizado mediante llamada a una API, la cuál proporciona imágenes según los parámetros proporcionados.

  • El juego Atlantis ha sido creado siguiendo un curso de Phaser en 2D de Údemy. Simplemente para ver y tener una nueva experiencia en la creación de juegos con otra tecnología.

  • El Slider de cartas responsive lo realicé para tener un poco de soltura utilizando recursos externos y tener una guía/ejemplo para en futuro.
